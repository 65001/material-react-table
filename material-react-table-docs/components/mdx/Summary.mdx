A fully featured [Material UI V5](https://mui.com/) implementation of [TanStack Table V8](https://tanstack.com/table/v8) (formerly called [React Table](https://react-table-v7.tanstack.com/)), inspired by [material-table](https://material-table.com/) and the [MUI X DataGrid](https://mui.com/x/react-data-grid/#mit-version), written from the ground up in TypeScript.
