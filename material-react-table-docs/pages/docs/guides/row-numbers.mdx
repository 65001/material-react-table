## Row Numbers Feature Guide

### Relevant Props

- enableRowNumbers
- rowNumberMode

---

There are two row number modes that you can enable.

### Enable Row Numbers (Original Mode)

In the default rowNumberMode (`original`), the row numbers linked to the original index of the data array. This means that when you search or filter, the same row numbers will stay with the row, though they will not be in the same order when sorted or will skip numbers when filtered.
