## Virtualized Example

Material React Table has a built-in row virtualization feature (via `@tanstack/react-virual`) that allows you to render a large number of rows without major performance issues.

Try out the performance of the table below with 50,000 rows! Filtering, Search, and Sorting also maintain usable performance too.
